# GoF Studio

Aplicacao educativa para estudar os 23 padrões GoF com:

- explicacoes didaticas em portugues
- exemplos de implementação em Java
- diagramas UML renderizados com Mermaid
- validação rigorosa de correspondencia UML <-> Java (simbolo a simbolo)
- filtros por categoria, pesquisa e progresso local
- mini-quiz para reforco de aprendizagem

## Como correr

Abre o ficheiro `index.html` no browser ou serve esta pasta com um servidor estatico.

Exemplo com Python:

```bash
python -m http.server 8080
```

Depois abre `http://localhost:8080`.
